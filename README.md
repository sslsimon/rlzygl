FTP程序更新服务配置
1运行ftpserv.exe
2建立用户GUEST，根目录为程序目录
3 有新的程序放到服务器端程序目录后使用Set_version.exe 上传新的修改日期到数据库做为版本更新的比较
4 Update.exe  为更新程序的守护进程。

Web安装步骤  
1 安装ib6odbc 驱动
2 双击导入注册表文件
3 执行WEB目录里的 Sws 2.3.exe 并建立建快捷方式到开始菜单的“启动”
4 本机浏览器地址栏输入127.0.0.1 可以显示用户登录页面表示安装成功
